How it works

Three steps, then you're done.

Pick a template, fill in the project details, and Hatch builds your entire folder structure — correctly named and ready to use.

1

Pick your template

Choose from templates you've built for your type of work — commercial, short film, social content, whatever fits your workflow.

2

Enter the project info

Type in the client name, project name, and shoot date. Hatch fills these into the folder and file names automatically.

3

Hit create

Your entire folder structure is built in milliseconds — exactly the same every time, no typing, no forgetting, no mess.

Features

Built around how editors actually work.

Build your own templates

Set up your ideal folder structure once and save it. Hatch uses it every project — sub-folders, naming, all of it.

Consistent naming, every time

Folders and root names auto-fill with {client}, {project}, and {date} — no more "Final_v2_FINAL" chaos.

Client list built in

Hatch remembers your clients. Type a name once and it's saved. You can also sync with a Google Sheet or import a CSV.

Toggle folders on or off

Not every project needs every folder. Quickly enable or disable parts of your template before you create.

Share templates across machines

Host your template on a private GitHub Gist. Any machine running Hatch picks it up automatically on startup.

See all your projects at a glance

Connect a drive and Hatch shows every project on it — with a health check to flag anything that looks incomplete.

Media Ingest

Copy cards with confidence.

The Ingest Hub moves footage from your cards to your project folder and checks that every file transferred cleanly. No silent failures, no missing files.

  • Checks file count and size before calling it done
  • Detects your connected drives and project folders automatically
  • One click to open any folder in Finder or Explorer
