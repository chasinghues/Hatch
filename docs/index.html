Technical Manual

Hatch is built for multi-seat facility integration. Maintain a single source of truth across your entire post-production floor.

GitHub Gist Sync

Host your production templates on a private or public GitHub Gist. Every edit workstation in your facility will automatically pull the latest structural schema on startup.

Google Sheets Client Hub

Stop typing client names manually. Connect a master Google Sheet to Hatch. Any new client added to the CRM is instantly selectable in the project creation menu.

Bulk CSV Importing

Migrate existing client databases or project lists instantly. Drag and drop any .csv to populate your local client registry without manual data entry.

Auto-Learning Registry

Type a new client or project type once, and Hatch remembers it forever. Your local database grows with your workflow, making recurring clients instantly accessible without setup.

Dynamic Filename Enforcer

Enforce strict naming conventions for the root project folder. Eliminate "Final_v2_FINAL" chaos by forcing a structured naming architecture from day one.

Multi-Drive Monitoring

Monitor multiple high-speed storage volumes. Hatch identifies existing projects and their "Health Status" based on structural integrity checks.
